# Unium — Contracts & Invoices on Ethereum

A web app for freelancers to generate contracts and pay invoices via Ethereum smart contracts.

![Stack](https://img.shields.io/badge/React-18-blue) ![Ethers](https://img.shields.io/badge/ethers.js-v6-purple) ![Vite](https://img.shields.io/badge/Vite-5-yellow) ![Solidity](https://img.shields.io/badge/Solidity-0.8.19-gray)

---

## Features

- **Multi-step project form** — collect freelancer/client info, deliverables, timeline, and fees
- **Auto-generated contract** — produces a full Freelance Service Agreement with industry-standard legal clauses (IP assignment, NDA, kill fee, force majeure, limitation of liability)
- **Invoice breakdown** — split by Full Upfront, 50/50, or Per Milestone schedule
- **Ethereum payments** — pay invoices directly from MetaMask; transactions link to Sepolia Etherscan
- **Smart contract included** — `contracts/FreelanceInvoice.sol` for on-chain invoice management (Solidity ^0.8.19)

## App Flow

```
Home → New Project (4-step form) → Contract View → Invoices → Pay via MetaMask
```

1. **New Project** — fill out 4 steps: Parties → Project Details → Payment Terms → Review
2. **Contract** — view and print the generated Freelance Service Agreement
3. **Invoices** — pay each invoice milestone by milestone using MetaMask

---

## Smart Contract

`contracts/FreelanceInvoice.sol` — deploy to any EVM chain (Sepolia, Mainnet, etc.)

```
createInvoice(freelancer, description, projectTitle, amountWei) → invoiceId
payInvoice(invoiceId) payable → transfers ETH to freelancer
getInvoice(invoiceId) → Invoice struct
```

The frontend currently uses **direct ETH transfers** (no contract deployment needed). To use the smart contract, deploy it and integrate the ABI from `src/abis/FreelanceInvoice.json`.

---

## Contract Legal Sections

The generated Freelance Service Agreement includes:

1. Scope of Work & Deliverables
2. Timeline
3. Compensation & Payment Schedule
4. Intellectual Property Assignment
5. Confidentiality / NDA
6. Revisions Policy
7. Termination & Kill Fee (20% of remaining balance)
8. Force Majeure
9. Limitation of Liability (capped at total project fee)
10. Governing Law
11. Signature Block with Ethereum addresses
